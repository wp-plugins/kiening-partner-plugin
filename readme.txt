The Kiening-Partner-Plugin is a Widget showing a banner link and / or a badge in everey post 
for the Kiening partner program and should be used only in german speaking regions.

== Description ==
Sie lesen die Datei readme.txt zum Kiening-Partner-Plugin
Das Kiening-Partner-Plugin bietet die Möglichkeit ein Banner-Widget auf Ihrer Wordpress Seite anzuzeigen.
Weiterhin können kleine Link-Logos auf jeder Seite, jedem Artikel eingeblendet werden.
Zur Verwendung des Plugins benötigen Sie eine Kiening-Partner-ID, die Sie unter http://www.kiening.eu/partner
beantragen können.

Credits:

== Installation ==

1. Unzip the package
2. Upload the kiening-widget directory to your plugins folder
3. Activate the plugin on the plugin screen.
4. Visit the widget page to place and customize the Kiening-Widget
